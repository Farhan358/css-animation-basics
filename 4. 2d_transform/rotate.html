transfom : rotate(X)
rotate function rotate the property clockwise or anti-clockwise based on the value of X. If X is positive rotation will be clockwise. Else anti-clockwise.
We can use 3 unit as value of X.
deg -> transfom : rotate(90deg); or transfom : rotate(-90);
turn -> 1 turn = 360deg
transfom : rotate(1turn); or transfom : rotate(-1turn);
gradient -> 400 grad = 1 turn
transfom : rotate(200grad); or transfom : rotate(-200grad);
